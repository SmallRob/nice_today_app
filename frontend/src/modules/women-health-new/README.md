# 女性健康管理模块 (新版本)

基于主应用的技术栈以及布局特点优化的女性健康管理模块。

## 功能特性

1. 完整的模块目录结构和类型定义
2. 预测算法工具和数据存储管理
3. 基于科学的预测经期周期和预测结果，显示计算和统计数据分析
4. 可视化日历组件和周期阶段展示
5. 基于ZenCalendar风格的现代化日历组件
6. 周期阶段可视化展示（经期、卵泡期、排卵期、黄体期）
7. 事件标记和健康记录显示
8. 响应式设计和暗黑模式支持

## 目录结构

```
women-health-new/
├── components/           # React组件
│   ├── calendar/         # 日历相关组件
│   ├── charts/           # 图表组件
│   ├── forms/            # 表单组件
│   └── ui/               # 通用UI组件
├── hooks/                # 自定义React Hooks
├── services/             # 业务逻辑和服务
├── store/                # 状态管理
├── types/                # TypeScript类型定义
├── utils/                # 工具函数
└── styles/               # 样式文件
```
