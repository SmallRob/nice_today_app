# 女性健康管理独立应用

这是一个基于 React 18 + Tailwind CSS + Electron 技术栈的独立女性健康管理应用。

## 技术栈

- **React 18** - 现代化前端框架
- **Tailwind CSS** - 原子化CSS框架
- **Electron** - 跨平台桌面应用框架
- **TypeScript** - JavaScript超集语言

## 功能特性

- 经期周期预测
- 健康数据分析
- 日历可视化展示
- 健康记录标记
- 本地数据存储
- 响应式设计
- 暗黑模式支持

## 开发环境搭建

```bash
# 安装依赖
npm install

# 启动开发环境
npm start
```

## 构建和发布

```bash
# 构建应用
npm run build

# 打包为可执行文件
npm run dist

# 打包为特定平台
npm run dist:mac
npm run dist:win
npm run dist:linux
```
